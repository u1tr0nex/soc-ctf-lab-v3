## Overview

This is an interactive **Security Operations Center (SOC) Capture The Flag (CTF)** lab designed to help students practice real-world incident response, investigation, and defensive decision-making.

The lab contains **4 independent challenges** that can be completed in any order.

## What You'll Do

Each challenge presents a realistic security scenario and a set of tasks. Your goal is to read the evidence carefully, investigate the situation, and choose the best response for each task.

### Challenge Topics
- USB drop attack investigation
- Drive-by download response
- DDoS mitigation
- Firewall rule investigation

## How the Lab Works

### Getting Started
1. Open the main hub page in your browser.
2. Use a **private/incognito window** if you want to start fresh.
3. Click any challenge card to begin.

### Playing the Challenge
- Each challenge has multiple tasks.
- Tasks **validate before unlocking** the next one.
- When you finish a challenge, you receive a **flag**.
- Copy the flag after completion.
- Progress is saved in your browser.

### Progress Reset
If you want to restart:
- Use **Reset challenge** inside the lab, or
- Clear site data for this folder, or
- Open the lab in a private/incognito window.

## Learning Objectives

By completing this lab, students practice:
- Identifying suspicious activity from logs and scenario context.
- Selecting appropriate containment and remediation actions.
- Recognizing common attack patterns.
- Understanding basic defensive response decisions.
- Applying incident response thinking to realistic security events.

## Quick Start for Students

1. Download or clone the repository.
2. Open `index.html` in a browser.
3. Click any challenge card.
4. Complete the tasks and copy the flag when done.

## Important Notes

- The challenges are browser-based and require no installation.
- Use a private/incognito window to start over quickly.
- If you previously started an older numbering set, clear site data to avoid stale progress.
